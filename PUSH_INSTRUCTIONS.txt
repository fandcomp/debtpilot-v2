TROUBLESHOOTING

  → Generate personal access token di GitHub settings

  "Build error"
  → Check Netlify deployment logs untuk error details

Application not loading:
  → Clear browser cache (Ctrl+Shift+Del)
  → Check F12 console untuk JavaScript errors
